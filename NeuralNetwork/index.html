Welcome to Neural Network’s documentation!¶

This is a pure Python implementation of a Feed-Forward/Back-Propagation (FFBP) Neural Network.

This allows a Neural Network to be created and subsequently trained on data to derive Perceptron weights and biases. The project test/ directory contains examples of this Neural Network training in action. If this NeuralNetwork module is not installed, the tests must be run from within the package directory, either in the package root or in the tests directory itself.

This project uses autogenerated Sphinx documentation. If all goes well, you should be reading that right now. If not, then check point your web browser of choice to docs/build/html/index.html in the package directory.

This project was developed using Python 3.6 to take advantage of the type annotations that have been introduced into the Python language. There is a good chance this will not work on older versions of Python.

There are two Homework examples that use the Neural Network to answer problems for EN.625.438/EN.605.447. There are other examples that derive a Neural Network capable of performing XOR and OR operations by using FFBP to train the network and applying threshold logic at the output layer of the Neural Network.
